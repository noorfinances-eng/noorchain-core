# NOORCHAIN Core

Core blockchain implementation for **NOORCHAIN**, a Swiss ethical blockchain
based on **Cosmos SDK + Ethermint (EVM compatible)** with **Proof of Signal Social (PoSS)**.

> This repository is focused on the technical implementation only (code, modules, configs).

## Status

- Phase 1 (Cadrage & décisions): ✅
- Phase 2 (Technical base & repo structure): ⏳ in progress

## Structure (work in progress)

- `cmd/noord/` — main entrypoint for the NOORCHAIN node binary
- `app/` — NOORCHAIN Cosmos/Ethermint application wiring
